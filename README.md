## 工程打开与使用

> 前置条件：正确安装VisualStudio用于编译UE工程

1. Clone或直接下载到本地目录。
2. 右键.uproject文件 -> Generate VisualStudio project files (生成.sln文件)
3. 打开.sln文件进入VisualStudio界面，工具栏Build/生成 -> 生成解决方案
4. 等待编译完成，关闭VisualStudio，双击.uproject打开进入UE。
